== Description ==

How to increase qualified traffic on internet blogs ? Creating networks between them ! This is what offers Clickalto.

Members of the Clickalto community install on their own blog a small panel called 'widget' which lists links of blogs dealing with the same theme. These other blogs offer a link back to the original blog. Objective: increase qualified traffic to blogs!

== Installation ==

1. Use WordPress plugin upload section or upload Clickalto files to the `/wp-content/plugins/` directory ;
2. Configure and activate the plugin through the 'Clickalto' menu in WordPress ;
3. Customize the location of your widget through the ‘Appearance > Widgets’ menu in WordPress ;
